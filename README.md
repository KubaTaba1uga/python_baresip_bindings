# Send message via CLI

## Installation

Clone repositories
```
git clone https://github.com/baresip/baresip.git
git clone https://github.com/baresip/re
```
Install re dependencies
```
apt-get install cmake openssl zlib1g zlib1g-dev libssl-dev
```
Go into re
```
cd re
```

Build re
```
cmake -B build
```
